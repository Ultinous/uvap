Multi Task Runner API (prototype)

The Multi Task Runner API microservice publishes an REST API through which can be accessed the Multi Task Runner functionality. The Multi Task Runner is a Java based project witch manage the different task instances.

Task

Each task is typically an algorithm. For example, it performs transactions on kafka topics, the results of which are also written to a kafka topic. Each task has a configuration scheme. Multiple instances of a task type can run in parallel with different configurations. In order to manage a Task type through the API, it is necessary to register that type in the API Properties.

Crowd Detection

Triggered whenever the number of detected people reaches or exceeds a specified value in a set of areas.

  • registration key: com.ultinous.uvap.mtr.task.crowddetection.CrowdDetectionTaskHandler
  • expected input topic schema: ultinous.proto.kafka.ObjectDetectionRecord
  • output topic schema: ultinous.proto.vms.alerting.Alert

Empty Area

Alarms will be triggered if the number of detected people becomes 0 from a non-zero value or becomes non-zero from 0.

  • registration key: com.ultinous.uvap.task.kafka.empty.area.EmptyAreaTaskHandler
  • expected input topic schema: ultinous.proto.kafka.ObjectDetectionRecord
  • output topic schema: ultinous.proto.vms.alerting.Alert

Wrong Direction

Triggered whenever a person moves to an unwanted direction.

  • registration key: com.ultinous.uvap.task.kafka.crowd.detection.WrongDirectionTaskHandler
  • expected input topic schema: ultinous.proto.kafka.PassDetectionRecord
  • output topic schema: ultinous.proto.vms.alerting.Alert

Service

Services that can be started during configuration time.

Heartbeat Service

Heartbeat Service inserts a new heartbeat record in a given topic from time to time.

  • registration key: com.ultinous.uvap.mtr.service.KafkaHeartbeatService

Storage

Configuration Storage module.

Kafka Storage

Kafka Storage is a Kafka based storage implementation.

  • registration key: com.ultinous.uvap.multi.task.runner.storage.KafkaStorage
